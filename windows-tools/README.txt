- 00-menu-rugbynow.bat

Comandos directos:
- 01-dev.bat
- 02-build-next.bat
- 03-deploy.bat
- 04-sync-matches.bat
- 05-sync-live.bat
- 06-sync-live-clock.bat
- 07-sync-live-finish.bat
- 08-sync-standings.bat
- 09-export-snapshot.bat
- 10-check-supabase.bat
- 11-rebuild-flashscore.bat
- 12-generate-team-profiles.bat
- 13-build-weekly.bat
- 14-lint.bat

Todos entran solos a la carpeta del proyecto y ejecutan el comando correcto.
